# XHX-data-sys

## 项目简介

XHX-data-sys 是一个用于分析电商零售数据的综合系统。项目使用 Hadoop 和 Spark 进行大规模数据处理和分析，后端使用 Flask 实现，并采用 ECharts 进行数据可视化展示。

## 功能特性

- **大数据处理**: 使用 Hadoop 和 Spark 对海量电商零售数据进行处理和分析。
- **后端服务**: 通过 Flask 提供数据接口和服务。
- **数据可视化**: 使用 ECharts 进行丰富的图表展示，直观呈现分析结果。
